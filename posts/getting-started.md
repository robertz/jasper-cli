The jasper-cli project is aimed at quickly generatng staic websites using CommandBox by Ortus Solutions. It is very easy to get started with commandbox-jasper.

``` bash
mkdir newproject
cd newproject
box jasper init
box jasper build
box server start 
box jasper watch

```
